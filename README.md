# 个人知识库驱动的深度研究助手
本项目构建了一个以个人知识库为核心的轻量级深度研究工具，旨在解决研究人员、学生和知识工作者在信息处理中面临的“知识碎片化”和“浅层检索”两大核心痛点。用户往往将文献、笔记、报告散落在不同文件中，传统全文搜索仅能返回关键词匹配的片段，无法揭示跨文档的深层关联、隐藏模式与研究空白。该助手通过本地智能索引与启发式多步分析，将静态文档集合转化为可交互、可推理的动态知识网络。

核心逻辑流：

文档摄取与知识索引

支持多种格式（PDF、Markdown、纯文本、HTML）的上传、URL抓取或手动粘贴。

对长文档进行自适应分块（带段落重叠），保留上下文连续性。

采用TF-IDF算法构建倒排索引，计算每个片段的稀疏向量表示，用于后续语义相似度计算。

多模式语义检索与推理链

接收用户自然语言查询，进行中文/英文混合分词，生成查询向量。

通过余弦相似度+关键词重叠加成，检索最相关的知识片段，并按去重策略精选Top-K结果。

非多Agent协作，但内置了单链式的多步分析流水线：

深度分析模式：从检索结果中提取关键概念，构建概念共现网络；自动识别跨文档的关联模式与矛盾点；生成研究建议，指出知识空白和潜在方向，模拟了“发现→关联→推演→建议”的推理链。

问答模式：定位最高置信度片段直接回答，同时标注支持性证据的数量，给出来源可信度。

摘要模式：聚合相关片段，生成结构化要点和阅读时间估算。

探索模式：意外关联发现——通过对比不同文档片段的关键词重叠度，挖掘非预期的主题连接，并输出概念网络和知识空白提示。

交互与反馈闭环

所有分析结果均可视化呈现（概念标签、来源文档、相关度分数），支持一键导出研究记录。

知识库可持续扩充，索引动态更新，所有数据存储在浏览器本地，保障隐私与离线可用性。

提供上下文感知的查询建议，引导用户深入探索未知领域。

技术亮点：

纯前端实现，零后端依赖，即时响应。

启发式推理引擎替代传统单步搜索，提供类“研究助理”的分析深度。

设计多种研究范式（深度、问答、摘要、探索），覆盖从具体提问到开放式探索的完整研究周期。

未来扩展方向：
可接入本地大语言模型（如Ollama）增强生成式回答，引入图神经网络实现真正的知识图谱推理，或采用多Agent框架（如文档分析Agent、假设生成Agent、批判性审查Agent）实现协作式深度研究。
